# Fake News Detector

A machine learning-based web application that detects whether a news article is Real or Fake using Logistic Regression and TF-IDF vectorization. The project includes a user-friendly interface built with Streamlit, prediction history, and dataset statistics.

---

## Features

- Real-time fake news prediction
- Fast model loading using Streamlit caching
- Clean and responsive user interface
- Supports both Light and Dark mode
- Prediction history tracking
- Dataset statistics page
- Confidence score for each prediction

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Logistic Regression
- TF-IDF Vectorization
